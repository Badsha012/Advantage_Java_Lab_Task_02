📐 Shape Area Calculator (OOP Java)This project is a simple Java implementation of Object-Oriented Programming (OOP) principles. It calculates the area of various geometric shapes (Rectangle, Triangle, Circle, and Square) using specific class structures, encapsulation (getters/setters), and methods.🚀 FeaturesRectangle: Calculate area using length and width.Triangle: Calculate area using base and height.Circle: Calculate area using radius ($\pi r^2$).Square: Calculate area using a single side.Encapsulation: All attributes are managed through getter and setter methods.🛠️ Project StructureThe project consists of the following classes as per the technical requirements:ClassAttributesKey MethodsRectanglelength, widthsetLength(), getArea()Triangleheight, basesetHeight(), getArea()CircleradiussetRadius(), getArea()SquaresidesetSide(), getArea()
